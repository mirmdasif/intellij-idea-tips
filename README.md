# intellij-idea-tips
Tips and Tricks For The Best Java IDE

## Tip 1 : List All The Files That Have Local Changes

```bash
ctrl + 9
```

## Tip 2 : Convert a String to Upper Case

```bash
ctrl + shift + u
```

## Tip 2 : To Navigate from The Current Tab to The Next or Previous Tab

```bash
alt + right
alt + left
a;t + tab
```
